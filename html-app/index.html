PDF 병합

여러 PDF 파일을 하나로 합칩니다. 드래그로 순서를 변경할 수 있습니다.

📎
PDF 파일을 드래그하거나 클릭하여 선택
여러 파일을 선택할 수 있습니다 (최대 100MB/파일)
병합이 완료되었습니다.

PDF 분할

PDF에서 원하는 페이지만 추출합니다.

✂️
PDF 파일을 드래그하거나 클릭하여 선택
최대 100MB
쉼표로 구분, 범위는 하이픈 사용 (총 ?페이지)
분할이 완료되었습니다.

페이지 회전

PDF 페이지를 원하는 각도로 회전합니다.

🔄
PDF 파일을 드래그하거나 클릭하여 선택
최대 100MB
'all' 또는 페이지 번호 (예: 1,3,5-7) (총 ?페이지)
회전이 완료되었습니다.

PDF 최적화

PDF 파일을 최적화하여 크기를 줄입니다. (Lossless 최적화)

📦
PDF 파일을 드래그하거나 클릭하여 선택
최대 100MB
최적화가 완료되었습니다.

PDF → 이미지

PDF 페이지를 이미지 파일로 변환합니다.

🖼️
PDF 파일을 드래그하거나 클릭하여 선택
최대 100MB
변환이 완료되었습니다. ZIP 파일로 다운로드됩니다.

PDF → PPT

PDF를 파워포인트 프레젠테이션으로 변환합니다.

📊
PDF 파일을 드래그하거나 클릭하여 선택
최대 100MB
PPT 변환이 완료되었습니다.
